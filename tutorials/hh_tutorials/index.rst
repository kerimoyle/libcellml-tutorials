==================================
libCellML Hodgkin-Huxley Tutorials
==================================

This set of tutorials takes the user progressively through different aspects of the libCellML functionality while building towards a model of Hodgkin and Huxley's squid axon activation.
It is aimed at users who are already familiar with the concept of CellML and modelling.

For more information on the theoretical background behind these steps, please see the :ref:`Theory pages<theory_index>`.
